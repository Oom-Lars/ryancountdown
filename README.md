A romantic countdown website themed around Divergent and The Fault in Our Stars, celebrating love and the journey to being together.

## Features

- **Countdown Timer**: Real-time countdown to December 20th, 2025
- **Love Notes**: Heartfelt messages expressing love and appreciation
- **Our Goals**: Bucket list of activities to do together (Sushi date, movie marathon, gaming)
- **Quotes**: Inspiring quotes from Divergent and The Fault in Our Stars
- **Interactive Activities**: 
  - Couple's Quiz
  - Daily Check-In
  - Memory Timeline
  - Shared Playlist
  - Future Vision Board
- **Love Reminders**: Pop-up messages every 15 seconds (last 5 seconds)
- **Cat Pets**: Interactive cats named Fluffy (grey) and Kai (tuxedo)

## Theme

- **Colors**: Purple (#6B46C1), White, Black
- **Fonts**: Playfair Display (serif) for headers, Inter (sans-serif) for body
- **Style**: Elegant, romantic, literary-inspired

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

### Love Notes
Edit `app/components/LoveNotes.tsx` to customize the love notes.

### Goals
Edit `app/components/Goals.tsx` to add or modify goals.

### Quotes
Edit `app/components/Quotes.tsx` to add more quotes.

### Love Reminders
Edit `app/components/LoveReminders.tsx` to customize reminder messages.

### Countdown Date
Edit `app/components/CountdownTimer.tsx` to change the target date.

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide React (icons)

## Deployment

Build for production:
```bash
npm run build
npm start
```
